# Capstone_Project_JantungIn

FrontEnd dengan Vuejs, serta untuk mempercantiknya dengan Tailwind, Gsap, dll.

## Project Setup

```sh
cd JantungIn-FE
```

Atau

```sh
cd JantungIn-FE && npm install
```

### Run Debugging

```sh
npm run dev
```

atau check lagi package.json

### Compile and Minify for Production

```sh
npm run build
```
